**Time:** 3 hours

**Benefits:**
✅ 10-20% CTR improvement from search results  
✅ Better keyword targeting signals  
✅ Faster crawling & indexing  
✅ Improved relevance scores

---

### Step 2.2: Schema Markup
**Why:** Rich snippets = higher CTR, better visibility

**What to do:**
Add schema markup (JSON-LD):
1. **Course Schema** - Course name, description, price, rating
2. **Organization Schema** - Company details
3. **Review Schema** - Testimonials & ratings
4. **Aggregate Rating** - Average course rating

**Example Code:**
```json
{
  "@context": "https://schema.org/",
  "@type": "Course",
  "name": "Digital Marketing Fundamentals",
  "description": "Learn SEO, Google Ads, Social Media Marketing",
  "provider": {
    "@type": "Organization",
    "name": "ATGIC",
    "url": "https://atgic.ca"
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.8",
    "ratingCount": "245"
  },
  "offers": {
    "@type": "Offer",
    "price": "197",
    "priceCurrency": "USD"
  }
}
```

**Tools:** schema.org, Google Rich Results Test

**Time:** 2 hours

**Benefits:**
✅ Rich snippets in search results  
✅ Higher click-through rate (CTR)  
✅ Better visibility for course schema  
✅ Improved AI search visibility
