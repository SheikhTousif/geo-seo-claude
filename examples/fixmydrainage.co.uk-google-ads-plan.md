# Google Ads Strategy — Fix my Drainage (Camden, London)

**Client:** Fix my Drainage — https://fixmydrainage.co.uk/
**HQ:** Camden, London NW8 0SR — Tel: 020 3488 5529
**Niche:** Emergency drainage, blocked-drain unblocking, drain repair, CCTV surveys, drain cleaning
**Geo:** London Borough of Camden + adjacent NW/N/WC/W1/EC1 postcodes
**Goal:** Generate qualified inbound calls and form leads at a target CPL of **£25–£35** for in-house jobs (and a secondary stream of overflow leads that can be resold to vetted local sub-contractors).

---

## 1. Strategic Snapshot

| Lever | Position |
|---|---|
| Primary KPI | Cost-per-Lead (CPL) — phone call ≥ 60s OR validated form |
| Secondary KPI | Booked-job rate (lead → dispatched engineer) |
| Buying intent | Very high — emergency, "now"-driven micro-moments |
| Conversion window | Minutes, not days. ~70% of leads convert on the same call. |
| Channel mix | (1) Local Services Ads — pay-per-lead; (2) Google Search — pay-per-click; (3) Performance Max — supporting layer once data exists |
| Geo strategy | Concentric — start tight on Camden postcodes, expand by performance |

**Why this niche wins on Google:** Drain emergencies happen at home, late at night, often on a phone, with zero brand loyalty. The first credible 24/7 number a stressed user sees gets the call. That's a paid-search game, not an SEO one.

      ├── AG: Brand — "fix my drainage"
      └── AG: Site visitors who didn't call (RLSA)

**Why this beats the reference plan's three-campaign structure:**
- Splits emergency vs general so emergency ads can use a *24/7* headline + sitelink without diluting daytime general clicks.
- One ad group per postcode lets every headline carry the area name dynamically.
- Brand-defence campaign protects the cheapest leads (~£0.30 CPC) from competitor poaching.
- LSA runs in parallel as a CPL-only channel — different economics, layered, not replaced.

### Match-type policy
- **Phrase** as the default — best balance of reach and intent in UK plumbing auctions.
- **Exact** only on top 5 converters once cost data exists.
- **Broad** only inside Smart Bidding (tCPA) campaigns after 50+ conversions.

### Bidding strategy roadmap
1. **Days 1–14:** *Maximize Clicks* with Max CPC cap of £18 (emergency) / £12 (general) / £14 (local) — to bootstrap data fast.
2. **Days 15–45:** Switch each campaign to *Maximize Conversions* once it has 15+ conversions.
3. **Day 45+:** *Target CPA* set at £30 for Search, £40 for PMax, then ratchet down £2 every 7 days while volume holds.

### Geo-targeting
- Primary radius: 4-mile bid-modifier ring around Camden NW8 (+30% bid).
- Secondary: 8-mile ring (+0%).
- Outer M25 north-central: -50% bid (only show if the core ring is exhausted).
- Use **Presence**, not Presence-or-Interest, so you don't bid on tourists searching "London plumber" from abroad.

### Dayparting
- Emergency campaign: 24/7, +20% bid 22:00–05:00 (peak panic, low competition).
- General campaign: 07:00–21:00 only — daytime decision-making.
- Saturday + Sunday: +15% across all campaigns.

## 5. Local Services Ads (LSA) — The Borough Multiplier

LSA sits *above* the standard Search results on mobile and is the cheapest qualified call source in the London plumbing/drainage space.

| Why LSA fits Fix my Drainage perfectly |
|---|
| Pay-per-**lead**, not per click — hard to overspend |
| The Google-Guaranteed badge boosts trust for emergency callers |
| 549 reviews at 4.9★ already qualifies — that score directly drives ranking inside LSA |
| Calls can be force-routed to 020 3488 5529 with one-tap dialling |
| Disputes refunded for spam / out-of-area / wrong-service leads |

**Setup checklist**
- [ ] Apply for **Google Local Services Ads — Plumber** category.
- [ ] Pass background check + insurance verification (PL £2m minimum).
- [ ] Connect Google Business Profile and import the 549 reviews.
- [ ] Set service area = London Borough of Camden + 3 mile buffer.
- [ ] Add jobs: *Drain System Repair, Drain System Installation, Sewer Services, Plumbing Repair, Plumbing Inspection, Toilet*.
- [ ] Weekly budget cap: start £300/week, scale on accepted-lead ratio > 70%.
- [ ] Dispute every off-territory or sales-call lead within 48h — protects CPL.
- [ ] Target CPL: **£20–£30**.

## 7. Landing-Page Brief

This is where the reference plan is right — landing-page quality is the single biggest CPL multiplier in this niche. The current homepage is good but mixes too many services. **Build a dedicated emergency landing page per postcode cluster** (e.g. `/emergency-drain-camden-nw1/`, `/emergency-drain-hampstead-nw3/`).

### Above the fold (the only thing 80% of users see)
- H1 with keyword + area: **"Emergency Drain Unblocking in Camden NW1 — Available 24/7"**.
- Tap-to-call **020 3488 5529** rendered as a **44×44 px button**, sticky on mobile scroll.
- Response promise: **"We arrive within 60 minutes — guaranteed."**
- 3-field form (Name · Postcode · Phone) — never more.
- Trust strip: Google Guaranteed badge, 4.9★ × 549 reviews, *No Call-Out Fee · No Fix No Fee*.

### Mid-page conversion blocks
1. **"Available right now"** live-status pill (green dot, dynamic JS clock — "Engineer free in NW1").
2. Embedded Google Map showing service radius around NW8 with pinned vans.
3. Three service-photo cards (real team, real equipment — no stock photography).
4. Pricing transparency block: "From £85 + VAT for blocked drain clearance".
5. 4–6 review cards pulled live from the Google Business Profile.
6. "What we'll do in the first 60 minutes" timeline.

### Schema & SEO hooks
- `LocalBusiness` + `Plumber` + `EmergencyService` schema (already present on the site — extend it per landing page).
- `FAQPage` schema covering "How fast can you get to NW1?", "Do you charge a call-out fee?", "Do you fix blocked drains at night?".
- `AggregateRating` schema linked to the 549 reviews.
- Hreflang n/a; canonical to self.

### Speed & UX
- Target Lighthouse mobile: **Performance ≥ 85, LCP < 2.5s**.
- One web font, max. The current site is loading both *Exo* and *Rajdhani* + *Inter Tight* — drop to one.
- Defer LiteSpeed CSS & lazy-load the gallery (already partially done).
- Click-to-call instrumented as a primary conversion.

> **Expected lift:** moving from a single homepage to postcode-specific landing pages typically lifts emergency-niche conversion rate from 4–6% to **10–14%**, which on the Realistic media plan would push CPL from £51 down to **£30–£35** without spending a penny more.

## 8. Conversion Tracking Stack

| Conversion | Source | Tooling |
|---|---|---|
| Phone call from ad ≥ 60 sec | Google Ads call extension | Native (free) |
| Phone call from website ≥ 60 sec | Dynamic number insertion | **CallRail** or **WhatConverts** (£40/mo) |
| Lead form submit | Fluent Forms (already on site) | GTM → GA4 → Google Ads import |
| LSA call accepted | LSA dashboard | Native |
| Booked job (offline) | CRM webhook (Pipedrive / HubSpot) | Offline conversion import via Google Ads API |

Always set the **call ≥ 60 sec** event as a *Primary* conversion, and form-submits as *Primary*. Mark page-view goals as Secondary so they don't bias Smart Bidding.

## 11. Risks & Mitigations

| Risk | Mitigation |
|---|---|
